# Sketcher SDK

SketchPro is a hosted web application created specifically for computer assisted mass appraisal (CAMA).

This SDK provides guides and examples for how to integrate the SketchPro with your CAMA software.

## Licensing

SketchPro requires a CAMA Cloud License to access. To request a trial key please [contact us](https://woolpert.com/products/dcs-a-woolpert-company/#dcscontact) or follow the instructions to recover your license credentials [here](https://auth.camacloud.com/info/forgot.aspx).

## Integration

Learn about the data format, [Sketch Data Schema](./docs/sds/guide.md).

Refer to the [integration guide](./docs/integration-guide.md) for detailed information about how to integrate your application with the Sketcher.
